# Portfólio de Desenvolvimento Full-Stack | Gerson Machado

## 1. Visão Geral do Projeto
Este projeto é uma Landing Page profissional desenvolvida como desafio prático do curso de Introdução à Programação Front e Back-End (20h). O objetivo é apresentar meu percurso formativo e competências técnicas, unindo 12 anos de experiência em gestão de TI e Automação Industrial com habilidades modernas de desenvolvimento. O foco principal foi a criação de uma interface semântica, responsiva e com lógica de validação avançada.

## 2. Tecnologias Utilizadas
* **HTML5 Semântico:** Utilização de tags estruturais (`header`, `nav`, `main`, `section`, `article`, `footer`) para garantir acessibilidade e SEO.
* **CSS3 Moderno:** Implementação de layout flexível com Flexbox e variáveis CSS, assegurando total responsividade em dispositivos móveis e desktops.
* **JavaScript (Vanilla JS):** Desenvolvimento de lógica para manipulação do DOM e gestão de eventos em tempo real.

## 3. Diferenciais Técnicos e Implementação
* **Validação Real-Time com RegEx:** Diferente de validações simples, este projeto utiliza Expressões Regulares (RegEx) para validar e-mails e nomes completos instantaneamente através do evento `input`. O usuário recebe feedback visual imediato (mudança de cores nas bordas), melhorando a experiência de uso (UX).
* **Destaque em Automação (Back-End):** O portfólio documenta projetos reais de alta complexidade, como a integração de dados financeiros da B3 via protocolo RTD (ProfitChart Pro) e automação de clipping digital utilizando a biblioteca Playwright em Python.
* **Arquitetura de Eventos:** O uso de `addEventListener` garante um código limpo e desacoplado, seguindo as melhores práticas de desenvolvimento atuais.

## 4. Instruções de Execução
Para testar a aplicação, basta manter os arquivos `index.html`, `style.css` e `script.js` no mesmo diretório e abrir o arquivo HTML em qualquer navegador moderno. A validação do formulário pode ser testada digitando dados nos campos de contato; o botão de envio só será processado se os padrões de RegEx forem satisfeitos.
